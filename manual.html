1. 시작하기

접속하면 시작 화면이 나와요. 난이도를 고르고 게임 시작을 누르면 선수 선택으로 넘어갑니다.

난이도문제 방식
1단계 · 뜻 고르기영어 단어를 보고 4개 보기 중 알맞은 뜻 선택
2단계 · 철자 1뜻을 보고 빈칸에 들어갈 철자 선택 (빈칸 적음)
3단계 · 철자 2뜻을 보고 빈칸에 들어갈 철자 선택 (빈칸 많음)

선수 선택 (총 10명)

BLACK TEAM 5명(포인트가드 #23 ~ 센터 #99)과 라이트닝·플레임·웨이브·맘바·드래곤 중에서 Player 1, Player 2가 차례로 고릅니다. 같은 선수를 둘 다 골라도 OK!

✏️
이름 바꾸기 — 선수 카드의 연필을 누르면 내 이름으로 바꿀 수 있어요.
📷
사진 넣기 — 카메라를 누르면 내 얼굴 사진을 선수에 등록! 슛 쏠 때 내 얼굴 선수가 뜁니다.

2. 기본 규칙 — 공격권을 잡아라!

진짜 농구처럼 공격권이 있어요. 슛을 쏘려면 연속 2문제를 상대보다 먼저 맞혀야 합니다.

점프볼경기 시작! 같은 문제가 양쪽에 떠요. 먼저 맞힌 쪽이 공격권 획득 🏀
공격 vs 수비각자 화면에 문제가 계속 나와요. 패널에 🏀 공격 / 🛡️ 수비 표시와 주황 테두리로 누가 공인지 확인! 보너스 찬스 중엔 🏀 공격×2처럼 남은 횟수가 보여요.
스틸수비 쪽이 먼저 맞히면 🖐️ 스틸! 득점 없이 공격권만 빼앗아요. (드리블 선수 팝업)
턴오버공격팀이 틀리면 💥 턴오버! 상대가 공격권을 가져가면서 2연속 슛 찬스를 얻어요 — 첫 골을 넣어도 공격권을 안 뺏기고 한 번 더 공격! (단, 그 사이 스틸당하면 보너스도 사라짐)
슛!공격권을 가진 채로 맞히면 골인! 득점 🎉 (덩크 세리머니 팝업) 득점하면 공격권은 상대에게 넘어갑니다.

점수

상황점수
일반 슛+2점
3연속 정답 후 슛+3점 (3-POINTER!)
⭐ 최다빈출 단어 찬스+3점
틀리면연속 기록 초기화 (점수 그대로)
공격 중에 틀리면💥 턴오버 — 상대에게 공격권 + 2연속 슛 찬스

먼저 목표 점수에 도달하면 승리! 승리 화면에서 MVP 선수가 트로피를 들어요 🏆

3. 특수 기술

🛡️
블로킹 — 상대 패널의 블록 버튼을 누르면 3초 챌린지 시작! 그 안에 상대가 문제를 맞히면 블록 실패 → 상대에게 자유투 +1점. 막으면 상대 리듬을 끊을 수 있어요. 타이밍이 생명!
⏸️
작전타임 — 팀당 3회, 10초간 게임이 멈춰요. 상대가 연속으로 맞히며 불붙었을 때 흐름 끊기용으로!

4. 온라인 대전 (폰 2대)

1두 명 모두 onrender.com 주소로 접속
2한 명이 방 만들기 → 화면에 뜨는 방 코드 확인
3다른 한 명이 방 참가 → 코드 입력 → 자동 시작!

※ 온라인 대전은 공격권 없이 각자 빨리 맞히는 스피드전이에요. 경기가 끝나면 팀 승패가 🏆 전국 랭킹에 자동 기록됩니다.

5. 협동 패스 모드 (2~5명)

대결이 아니라 한 팀이 되어 함께 단어를 완성하는 모드!

1onrender.com 접속 → 👥 협동 만들기 → 코드를 친구들에게 공유
2친구들은 👥 협동 참가 → 코드 입력 (최대 5명)
3내 차례에 철자 하나를 맞히고 → 다음 사람에게 패스!
4자신 있으면 🙋 나한테 패스! 버튼으로 손들기 — 손든 사람에게 우선 패스

6. 랭킹 · 팀 관리

🏆
랭킹 — 시작 화면의 랭킹 버튼에서 전국 팀 순위(승/패)를 볼 수 있어요. 온라인 대전에서 이기면 자동 기록!
🏷️
팀명 바꾸기 — 팀 버튼에서 레드팀/블루팀 이름을 우리 팀 이름으로 바꿔보세요. 시즌 전적도 함께 기록됩니다.

7. 문제가 생겼을 때

업데이트했는데 예전 화면이 보여요 → 화면을 아래로 당겨 새로고침하거나, 브라우저를 완전히 닫고 다시 열어보세요.
소리가 안 나요 → 폰이 무음 모드인지 확인! 화면을 한 번 터치한 뒤부터 소리가 나기 시작해요.
방 만들기 버튼이 없어요 → vercel.app 주소에서는 온라인 기능이 꺼져 있어요. onrender.com 주소로 접속하세요.
"방이 없어요"라고 떠요 → 코드를 다시 확인하거나, 방을 새로 만들어서 다시 시도해 보세요.
